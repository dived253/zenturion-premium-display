# ZENTURION Premium Car Rental

## Project Overview

Premium luxury car rental service display website for ZENTURION - offering exclusive vehicles including Rolls-Royce, Lamborghini, Mercedes G-Class, and Audi RS6 in NRW, Germany.

**Live Site**: https://zenturion.autovermietung.de

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Local Development

To run this project locally:

```sh
# Clone the repository
git clone https://github.com/dived253/zenturion-premium-display.git

# Navigate to the project directory
cd zenturion-premium-display

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Build for Production

Build the project for production:

```sh
npm run build
```

## Deployment

The site is deployed at: https://zenturion.autovermietung.de
